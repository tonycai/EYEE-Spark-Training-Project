== The Cardinal Rules ==
* 不允许出现图片外链，会存在潜在的风险。
* 批量修改超过100行数据，需要得到CTO的批准。
* 性能问题，核心接口响应时间要在1秒以内。
* 一次导出超过100条数据记录，需要得到CTO批准。
* 不批准连续超过两天的假期
* 添加定时任务，需要通过运维工程师上线流程，CC给技术主管和CTO。【NEW】
